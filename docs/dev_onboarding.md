# Developer Onboarding

This guide helps new developers get started with the organization's development practices.

## Prerequisites

- Git configured with your credentials
- Access to organization repositories
- Required development tools installed

## Getting Started

1. Clone the Agent State repository
2. Review the `AGENTS.md` files in relevant directories
3. Set up required environment variables
4. Run example scripts to verify setup

## Environment Variables

Common environment variables used across scripts:

- `DEVIN_API` - Devin API authentication token
- `JIRA_AUTH` - Jira authentication (or `JIRA_EMAIL` + `JIRA_API_TOKEN`)

## Next Steps

- Explore available scripts in `scripts/`
- Review organizational memory in `memory/`
- Check examples in `examples/`
