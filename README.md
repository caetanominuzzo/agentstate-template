# Agent State Template

This is a template repository for creating your organization's Agent State repository. Agent State provides a persistent memory layer for AI agents, enabling them to store tools, scripts, organizational context, and knowledge that persists across sessions and even across different agents.

## Getting Started

1. Fork this repository to create your organization's Agent State repository
2. Update this README with your organization's specific information
3. Add your agents to the repository and configure access
4. Start storing tools, scripts, and organizational memory

## Structure

- `scripts/` - Helper scripts, API clients, and automation tools
- `memory/` - Organizational memory (developers, PMs, systems)
- `docs/` - Documentation and onboarding guides
- `context/` - Organizational context and domain knowledge
- `templates/` - Reusable templates
- `examples/` - Usage examples
- `exchange/` - Small tracked artifacts for sharing
- `workspace/` - Temporary files and scratch work
